# По Барабану — Школа барабанов (baraban)

## Что это
Сайт школы барабанов для преподавателя Макса. Цель — получать заявки на пробное занятие (5-10 в неделю, конверсия 10%). Заявки уходят в Telegram.

## Стек
- **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS 3.4
- **Шрифты:** Outfit (заголовки), Inter (текст)
- **Изображения:** Sharp, next/image (lazy-load, WebP)
- **Деплой:** Standalone (Docker) на российском VPS
- **Аналитика:** Yandex Metrica

## Дизайн
- Тёмная тема, чёрно-оранжевая палитра
- Современный, дерзкий стиль
- Множество CTA-кнопок "Записаться" по всему сайту
- Анимации при скролле (AnimateOnScroll)
- CSS-переменные для цветов в globals.css

## Структура страниц
- `/` — Главная (Hero + секции)
- `/prices` — Цены
- `/teachers` — Преподаватели
- `/gallery` — Фото и видео
- `/reviews` — Отзывы
- `/contacts` — Контакты (карта, адрес, парковка)
- `/admin` — Админка (защищена паролем)

## Данные
- `data/prices.json` — цены и тарифы
- `data/teachers.json` — преподаватели
- `data/reviews.json` — отзывы
- `data/site.json` — общие настройки
- `lib/constants.ts` — константы (контакты, навигация)
