# По Барабану

Сайт школы барабанов. Да, название — каламбур, и нам не стыдно.

Задача простая: человек заходит, видит что тут круто, записывается на пробное за 500 рублей. Заявка улетает в Telegram. Профит.

## Что тут

Лендинг на Next.js 14 с тёмной темой и оранжевыми акцентами. Выглядит дерзко, как и должна выглядеть школа барабанов.

7 страниц: главная, цены, преподаватели, галерея, отзывы, контакты и админка для управления контентом.

Форма записи кидает заявку через Telegram Bot API — никаких баз данных, никакого оверинжиниринга. JSON-файлы для хранения данных, потому что для 10 заявок в неделю этого более чем достаточно.

## Стек

Next.js 14, TypeScript, Tailwind CSS, Sharp. Шрифты Outfit и Inter. Аналитика через Yandex Metrica. Деплой на VPS через PM2 + Nginx.

## Структура

app/ — страницы и API роуты
components/ — компоненты (Header, Footer, Hero, форма записи и всё остальное)
data/ — JSON с ценами, преподами, отзывами
lib/ — константы и хелперы

## Как запустить

```
npm install
cp .env.example .env.local
```

Заполнить .env.local (там всё подписано) и:

```
npm run dev
```

## Деплой на VPS

Раскомментировать `output: "standalone"` в next.config.mjs, дальше:

```
npm run build
node .next/standalone/server.js
```

PM2 чтобы не падало, Nginx чтобы проксировать. Классика.
